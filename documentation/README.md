# Documentation

Ce dossier contient la documentation technique du projet SEC-OPS LAB.

## Fichiers

- `DAT-SEC-OPS-LAB.tex` — Dossier d'Architecture Technique (source LaTeX)

## Compiler le DAT en PDF

1. Va sur [Overleaf](https://overleaf.com)
2. Créer un nouveau projet → Upload → dépose le fichier `.tex`
3. Clique sur **Compile** → télécharge le PDF

## Contenu du DAT

1. Présentation du projet et contexte CHU
2. Architecture du laboratoire virtuel
3. Description des outils (Ansible, OpenSCAP, Wazuh)
4. Plan de mise en œuvre et planning
5. Livrables et jalons
6. Clauses de sécurité et confidentialité
